# Drum Kit Fun Project

A basic Drum Kit web app built with HTML, CSS, and JavaScript. Press the keys or click the buttons to play different drum sounds.

## Live Demo
Try it here: [Drum Kit Live](https://drumkitprojectbysifat.netlify.app/)

## Features
- Play drum sounds using keyboard keys
- Click buttons to play sounds
- Simple and responsive design

## Tech Stack
- HTML
- CSS
- JavaScript
